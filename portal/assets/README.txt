Drop the BigMint logo here as:  bigmint_logo.png

The top bar uses it automatically (white logo on the blue bar works best, e.g. a
transparent PNG ~30px tall). If the file is absent, the portal falls back to a
"BigMint" text wordmark.

Co-brand (BigMint × Adani):  adani_logo.png

The top bar shows the Adani logo in a white chip next to the BigMint logo. Use a
tightly-cropped PNG (transparent or white background), ~26px tall — avoid extra
whitespace padding or the wordmark renders tiny. If the file is absent, the portal
falls back to a gradient "adani" text wordmark.
